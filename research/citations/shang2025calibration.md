# shang2025calibration — balance-penalized loss for PS estimation

Shang, Y., Chiu, Y.-H., & Kong, L. (2025). Robust propensity score estimation
via loss function calibration. *Statistical Methods in Medical Research*,
34(3), 457-472. DOI: 10.1177/09622802241308709

**Verified:** Run 1 agent (Crossref, 2026-07-18); re-verified via Crossref API
at bib entry (2026-07-19).

**Key facts:** adds a covariate-imbalance penalty to the training loss of
parametric and NN propensity models; under misspecification the calibrated NN
gave least-biased, lowest-variance ATEs.

**Used for:** Lit Review (the balance-aware training direction that our
prediction-trained baseline motivates); Discussion.
